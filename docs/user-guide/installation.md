### Install using pip:

```bash
pip install toggl-api-wrapper
```

### Or if using SQLite cache:

```bash
pip install "toggl-api-wrapper[sqlite]"
```

## Dependencies

### Main

- [HTTPX](https://www.python-httpx.org) - _For requests_

### Optional

- [SQLAlchemy](https://www.sqlalchemy.org) - _For Sqlite cache_
